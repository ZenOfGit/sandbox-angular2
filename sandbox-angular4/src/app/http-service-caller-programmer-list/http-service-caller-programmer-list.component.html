Here, we will use the database instead of the hard-coded data from the get method.
Our Angular service will call the Web API service which will fetch data from SQL Server.
We need to import HttpModule in our root module first, then import Http in our service (then inject it.)

Steps in our service:

  • 1. Return the get method of the Http service
  • 2. Import Observable from rxjs
  • 3. Specify the url to which we want to issue the get request
  • 4. Change our return type from an array of our IProgrammer objects to an Observable of IProgrammer objects
  • 5. Import Map
  • 6. Chain the Map operator to our get method
  • 7. Specify the parameter that we want to pass which we will call response
  • 8. Import Response from @angular/http
  • 9. Set the type of our response parameter that we are sending to the arrow function to Response
  • 10. Set the arrow function to convert this response to an array of IProgrammer objects
  • 11. Get the .json out of the response
  • 12. Use the type casting operator on it to cast it to an IProgrammer array

Steps in our list component to subscribe to the observable:

  • 1. Chain the subscribe() method onto the getEmployees() method
  • 2. Use an arrow function that will receive programmer data so we will name it programmerData
  • 3. Use the arrow and code the function to set this programmers equal to whatever data the callback receives from this Observable, i.e. programmerData
  • 4. Get rid of the programmers before the equal sign (if we have it leftover from using an alternative method to get the data)

  • add name="Access-Control-Allow-Headers" value="Content-Type
  • add name="Access-Control-Allow-Methods" value="GET, POST, PUT, DELETE, OPTIONS
