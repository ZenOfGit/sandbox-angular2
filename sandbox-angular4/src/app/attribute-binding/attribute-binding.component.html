We use the corresponding DOM objects of HTML elements to modify them.
The DOM contains teh HTML elements as objects including properties, methods, and events.
The DOM is a standard for accessing, modifying, adding, or deleting HTML elements.
We can use interpolation and property binding to bind component class properties to HTML DOM element properties, and not attributes.
There are situations when we want to be able to bind to HTML element attributes and not properties.
The colspan element does not have a corresponding DOM property, so we want to bind to it’s HTML element attribute directly.
Angular recommends using property binding whenever possible, but it has provided the ability to use attribute binding.
In the class, we have hard coded the colspan property to 2. Attempting to use interpolatin would give a "Can't bind" error.
Property binding would give the same error because colspan does not have a DOM proerty, so we must use attribute binding.
